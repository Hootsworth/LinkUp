# Security Policy

Because LinkUp handles peer-to-peer screen capture, clipboard syncing, and mouse/keyboard event injection, we take security extremely seriously.

## Reporting a Vulnerability

**Please do not report security vulnerabilities through public GitHub Issues.** 

Instead, report security vulnerabilities by emailing the maintainers directly or utilizing GitHub's private vulnerability reporting feature on the repository.

Once a report is received:
1.  We will acknowledge receipt of your report within 48 hours.
2.  We will work on a fix as quickly as possible.
3.  We will coordinate a release and publish a security advisory alongside the patch.

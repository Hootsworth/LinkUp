P2P Remote Desktop

LinkUp enables direct, secure, and lightning-fast desktop sharing and input control across networks. Fully open-source and free.

Host Local Sessions

Activate Host Mode to stream your screen. It creates a secure 6-digit connection code that remote users can use to view and control your desktop.

Remote Control

Viewer Mode

Type in the host's 6-digit pairing code to view their screen. Moves, clicks, and keys are captured on your image viewport and executed on the host.

Peer-to-Peer

WebRTC & TURN Fallback

Connections use public STUN servers to bypass NAT routers. We've added TURN server fallback configurations so screen sharing works even behind symmetric NATs.
